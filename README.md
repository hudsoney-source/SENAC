# SENAI

Repositorio com atividades, exercicios e projetos desenvolvidos ao longo do curso.

## Estrutura

- `BI/`: estudos e exercicios com `pandas`, `numpy` e `streamlit`
- `Big Data/`: atividades de extracao de dados, SQL e analise
- `IOT/`: exercicios com ESP32 e simulacoes no Packet Tracer
- `NoSQL/`: exemplos com Docker, PostgreSQL, MongoDB e Redis
- `PI/`: projeto integrador de deteccao de fadiga com IoT e IA

## Tecnologias

- Python
- Streamlit
- Pandas
- NumPy
- Docker
- Redis
- MongoDB
- ESP32

## Como executar exemplos em Python

1. Crie ou ative um ambiente virtual.
2. Instale as dependencias:

```bash
pip install -r requirements.txt
```

3. Execute o script desejado dentro da pasta correspondente.

## Projeto em destaque

O diretorio `PI/` concentra a documentacao e os arquivos principais do sistema de deteccao de fadiga, incluindo firmware, arquitetura e prototipos de IA.
